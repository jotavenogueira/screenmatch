# 🎬 ScreenMatch

Aplicação em Java que utiliza **requisições HTTP** para buscar informações sobre filmes e séries e exibi-las de forma organizada no console.

---

## 📌 Descrição

O projeto realiza chamadas HTTP para um serviço externo e interpreta a resposta recebida em formato JSON.  
O foco é demonstrar:

- Uso do `HttpClient` nativo do Java
- Leitura e manipulação de JSON
- Organização de código em camadas simples

Não há integração completa com API (como autenticação, múltiplos endpoints ou filtros avançados). O objetivo é apenas **fazer a requisição**, **receber os dados** e **exibi-los**.

---

## 🚀 Funcionalidades

- 🔗 Envio de requisições HTTP GET
- 📥 Recebimento e leitura de JSON
- 📝 Conversão dos dados para objetos do sistema
- 📽️ Exibição de informações sobre filmes e séries
- ⭐ Avaliação e cálculo de média
- 🗂️ Estrutura básica de POO
